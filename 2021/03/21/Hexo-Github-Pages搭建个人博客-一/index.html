Hexo+Github Pages搭建个人博客(一)

安装hexo

  1. 安装Node js和npm并配置环境变量，官网地址https://nodejs.org/en/download/

    • npm换源

  2. 安装git并注册、在github中添加自己的SSH Key

  3. 安装并初始化hexo

  4. 常用hexo命令

配置Hexo

  1. 下载主题Butterfly

  2. 修改配置文件_config.yml，将主题改为butterfly

  3. 修改配置文件_config.yml，连接到建立好的github pages中

  4. 安装pug和stylus的渲染器

  5. 把./themes/butterfly目录下的 _config.yml 复制到博客根目录下，并重命名为_config.butterfly.yml

  6. 修改配置文件_config.butterfly.yml

    • 官网上有一些配置教程，见 https://butterfly.js.org/
